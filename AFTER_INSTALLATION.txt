================================================================================
TROUBLESHOOTING
================================================================================

Application Won't Start:
   ✓ Verify Python 3.10+ is installed
   ✓ Check that MetaTrader 5 is installed
   ✓ Run as Administrator
   ✓ Check Logs/ folder for error messages
   ✓ Contact support: support@aventaai.com

Can't Connect to MetaTrader 5:
   ✓ Ensure MT5 is running and logged in
   ✓ Verify account credentials
   ✓ Check internet connection
   ✓ Disable antivirus temporarily
   ✓ Verify firewall isn't blocking MT5

Telegram Notifications Not Working:
   ✓ Verify bot token is correct
   ✓ Check Telegram User ID configuration
   ✓ Test connection in Settings
   ✓ Verify internet connection
   ✓ Check firewall/proxy settings

Backtest Results Seem Incorrect:
   ✓ Verify MT5 data is complete (download historical data)
   ✓ Check date range selection
   ✓ Verify indicator settings
   ✓ Review trade conditions in logs
   ✓ Export results to CSV for manual review
